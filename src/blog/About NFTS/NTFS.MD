# 概述

  NTFS（New Technology File System，新技术文件系统）是微软为Windows操作系统开发的主流文件系统，提供高级功能，如文件加密、安全描述符、磁盘配额、大文件支持和日志记录，相比旧的FAT文件系统更安全、更可靠，支持大容量文件和高性能.

- 可以处理大容量(高达2^64GB存储单位)
- 内置数据压缩功能
- 它使用日志文件来记录事务（操作
- 字节序: 磁盘上所有的数据都是小端字节序

使用 NTFS 文件系统格式化卷会导致创建多个系统（元数据）文件，例如 $MFT — 主文件表、$Bitmap、$LogFile 等，其中包含有关 NTFS 卷上所有文件和文件夹的信息。

NTFS 文件系统还支持数据访问控制和所有权权限，这对于关键数据的完整性至关重要。虽然在 Windows NT 计算机上共享的文件夹会被分配特定的权限，但 NTFS 文件和文件夹无论是否共享，都可以分配权限。

NTFS 文件系统设计简洁却功能强大。基本上，卷上的所有内容都是一个文件，而文件中的所有内容都是一个属性，包括数据属性、安全属性和文件名属性。NTFS 卷上分配的每个扇区都属于某个文件。甚至文件系统元数据（描述文件系统本身的信息）也是文件的一部分。
