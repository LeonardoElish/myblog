```c
// 内存序选项
std::memory_order_relaxed    // 最弱的顺序保证，只保证当前操作原子性
std::memory_order_consume    // 数据依赖顺序(C++17后不推荐使用)
std::memory_order_acquire    // 获取操作，保证后续读取不会重排到此操作之前
std::memory_order_release    // 释放操作，保证之前的写入不会重排到此操作之后
std::memory_order_acq_rel    // 获取和释放的组合
std::memory_order_seq_cst    // 最强的顺序保证，默认选项

// 使用例子
a.load(std::memory_order_acquire);
a.store(10, std::memory_order_release);
a.fetch_add(5, std::memory_order_acq_rel);
```

- `compare_exchange` 有 `success` 和 `failure` 两个内存序参数。
    
- 约束：`failure` 的语义**不能比**`success` 更强（failure ≤ success）。例如 `success=acq_rel, failure=acquire` 是允许的；`failure` 不能是 `release`。
    
- 如果你不熟悉内存序，使用默认的 `std::memory_order_seq_cst` 总是正确的（但可能比必要的更慢）。
