# 创建链接
创建链接可以通过 dos 命令 mklink 或者 powershell 中的New-Item 创建。  

mklink的语法如下:
```
C:\Users\you>mklink
创建符号链接。

MKLINK [[/D] | [/H] | [/J]] Link Target

        /D      创建目录符号链接。默认为文件
                符号链接。
        /H      创建硬链接而非符号链接。
        /J      创建目录联接。
        Link    指定新的符号链接名称。
        Target  指定新链接引用的路径
                (相对或绝对)。

C:\Users\you>
```

# 几种链接的区别

与 Linux 的文件系统中的 inode 与 block 类似，在 NTFS 文件系统中数据对象也赋予了独一无二的**文件 ID** 以及**与之对应的文件路径**，文件路径和文件 ID 对应，文件 ID 和数据对象绑定，最终才呈现为可供用户打开、编辑的文件。
